Docker no Windows 10

windows e linux containers no windows 10

Posted by Jair Nunes on May 30, 2018

 

Passos para a instalacao do docker e e o uso do recurso de contêiner do Windows no Windows 10 Professional ou Enterprise (Anniversary Edition). Se você precisar se familiarizar com os contêineres, poderá encontrar essas informações em Sobre contêineres

 

Pre requisitos:

  •  Windows 10 Professional ou Enterprise (64bits).
  •  Atualizacao do Windows 14393.222 ou superior 
  •  Virtualização habilitado em seu computador
  •  Hyper-v habilitado

 

Baixe o docker e siga os passos de instalacao

PS: Após instalar o docker reinicie o computador para permissoes em diretorios e path
